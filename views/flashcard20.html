Pronombres demostrativos en inglés

¿Qué son los pronombres demostrativos?

Los pronombres demostrativos se utilizan para señalar o identificar personas, objetos o ideas según su distancia respecto al hablante y su número. Sustituyen al sustantivo y evitan repeticiones.

Tabla 1 — Pronombres Demostrativos

Pronombre Distancia Número Traducción Ejemplo
This Cerca Singular este / esta This is my favorite book.
These Cerca Plural estos / estas These are delicious.
That Lejos Singular ese / esa That was amazing!
Those Lejos Plural esos / esas Those are very expensive.

Tabla 2 — Reglas Clave

Regla Explicación Ejemplo
Sustituyen al sustantivo No van acompañados de un nombre ✓ This is mine / ✗ This book is mine (adjetivo, no pronombre)
Indican distancia Cerca (this/these) o lejos (that/those) This is closer / That is farther away
Indican número Singular (this/that) o plural (these/those) This one / These ones
No cambian por género Mismo pronombre para masculino, femenino, neutro This (él/ella/ello)
Usos adicionales Se usan para ideas, situaciones o hechos mencionados That's a good idea / This is important

Tabla 3 — Ejemplos de Uso

Pronombre Contexto Ejemplo
This / These Objetos cercanos al hablante This is my pen. These are my books.
That / Those Objetos lejanos al hablante That is his house. Those are their cars.
This / That Tiempo o situación This year was good. That was in 2020.
This / That Ideas o conceptos abstractos This is wrong. That doesn't make sense.
